前言
===
JQFramework：有效提高开发效率，一些网络，扩展，分类，uikit下的封装！ 

### 目录:


## Installation

#### From CocoaPods

```
pod  "JQFramework"
```

#### Manually 
- Drag all source files under floder `JQFramework` to your project.

## Usage

引入头文件: `#import "JQFramework.h"` 其他自行按需使用即可;
